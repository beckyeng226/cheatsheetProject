CSS Selectors & Specificity Cheatsheet

Selector Type Specificity Example
Element Applies to all elements h1 {}
Class Applies to all members of a class .class {}
ID Applies only to the item designated by the ID (and only used once) #id {}
!important Supercedes all other styling - useful when using multiple stylesheets styling !important
Chaining Selectors Combining multiple selectors - selects only elements that have that class h1.special {}
Nested Elements Applies to elements nested within a class .class li {}
